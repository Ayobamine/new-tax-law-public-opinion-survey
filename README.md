# New Tax Law Public Opinion Survey Analysis

## Project Overview

This project presents findings from a nationwide public opinion survey assessing Nigerians' awareness, understanding, and information sources regarding the proposed New Tax Policy.

The objective of the study was to evaluate public awareness levels, identify major communication channels through which citizens receive information about the policy, and understand citizens' perceptions of the proposed reforms.

## My Role

As a Data Analyst and Research Specialist, I was responsible for:

## Methodology

* Nationwide quantitative survey
* Representative sample of adult Nigerians
* Structured questionnaire
* Demographic weighting applied where necessary
* Analysis conducted using statistical and visualization tools
